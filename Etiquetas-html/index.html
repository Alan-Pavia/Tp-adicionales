Etiquetas de HTML

Etiqueta "Details y Summary"

Función: Crea elementos desplegables (como acordeones) sin necesidad de JavaScript.

Útiles para mostrar/ocultar contenido, como FAQs, mejorando la organización de información en la página.

Etiqueta "abbr"

Función: Proveer una abreviatura o acrónimo con su definición al pasar el mouse.

Aclara siglas o términos técnicos al mostrar su significado al pasar el cursor, ideal para textos educativos.

La OMS es una agencia de la ONU.

Etiqueta "datalist"

Función: Proveer una lista de opciones sugeridas para un campo de entrada.

Ofrece sugerencias al escribir en un campo, perfecto para búsquedas o formularios interactivos.

Etiqueta "progress"

Función: Indicar el progreso de una tarea (como una barra de carga).

Muestra el avance de tareas como cargas o descargas, ayudando al usuario a visualizar el progreso.
